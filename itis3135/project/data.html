🔍 Quick Lesson

In Python, variables are used to store data, and data types define what kind of data you are working with.

  • int: whole numbers, e.g., x = 5
  • float: decimal numbers, e.g., pi = 3.14
  • str: text, e.g., name = "Alice"
  • bool: True or False values, e.g., is_on = True

Python is dynamically typed, meaning you don’t have to declare the type—it figures it out for you!
